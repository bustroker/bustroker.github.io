Deploy dotnet core app to heroku.

2020-08-08

A dotnet core web app/api app is assumed to have been implemented already. Following is how to deploy it.

Heroku PORT

Deploying default dotnet app to heroku fails with error

It’s b of this issue related to dotnet core app using port 80. So the port needs to be changed to the one heroku sets in the environment variable PORT.

Dockerfile

Build image, push and deploy to heroku using heroku cli and heroku containers registry.

An app is assumed to have been created in heroku site.
Deployment can be done, as heroku explaines in their documentation, in various ways. Here’s one of them, using heroku container registry.

First install heroku cli

Login to heroku and heroku containers registry, using command line

Have heroku build and push the image to their registry

Deploy a container from the image pushed

Browse to (or curl) https://APP-NAME.herokuapp.com/.
