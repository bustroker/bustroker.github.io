Recover lost `PATH` variable

The idea is to open a cmd in the context of an application that was running before the PATH variable was broken, e.g., Chrome. Go to Chrome, hit ctrl+o, and it opens an explorer window. Go to the window’s path bar and type cmd and enter. A cmd opens “in the context” of Chrome. Now just display the PATH variable that was in place when Chrome was executed

echo %PATH%
