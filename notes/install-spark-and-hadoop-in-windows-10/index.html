Install spark to run locally

Versions

  • Java

OpenJDK Runtime Environment (AdoptOpenJDK)(build 1.8.0_292-b10)

  • Scala
sbt
console
Welcome to Scala 2.12.13 (OpenJDK 64-Bit Server VM, Java 1.8.0_292).

  • Spark Download from https://spark.apache.org/downloads.html Spark release 3.1.2 with hadoop2.7 folder downloaded and copied to C:\Spark\spark-3.1.2-bin-hadoop2.7. Add C:\Spark\spark-3.1.2-bin-hadoop2.7 to user’s PATH

  • Hadoop Downloaded from https://hadoop.apache.org/release/2.7.7.html There’s hadoop-2.7.7 folder downloaded and placed in C:\Hadoop\hadoop-2.7.7. There’s system variable HADOOP_HOME=C:\Hadoop\hadoop-2.7.7 There’s %HADOOP_HOME%\bin added to system’s PATH.
