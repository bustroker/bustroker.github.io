Git Stash and Diff Patches

Basic Stash Operations

git stash list

## Create stash
git stash 
## or with description
git stash push -m "description"

## Apply latest stash (stash@{0})
git stash apply 
## or specific stash
git stash apply [INDEX]

## Apply and remove latest stash
git stash pop

## Remove stash
git stash drop [INDEX]

Named Change Sets with Diff Patches

Stashes can’t be referenced by name, only by index. To save specific changes with identifiable names (useful for build configs or feature variants):

## Save working changes to a file
git diff > changes.save

## Re-apply later
git apply changes.save
