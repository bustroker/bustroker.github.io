Running C# Scripts with dotnet-script

Goal

Run C# scripts (.csx files) with NuGet package support using dotnet-script.

Install dotnet-script

dotnet tool install -g dotnet-script

Create a .csx script file

// script.csx
##r "nuget:Newtonsoft.Json, 13.0.1"

using Newtonsoft.Json;
using Newtonsoft.Json.Linq;

var args = Environment.GetCommandLineArgs();

if(args.Any(a => a == "--dry-run"))
{
    return;
}

Console.WriteLine("Using Newtonsoft.Json to parse JSON.");

var json = JObject.Parse(@"{ 'name': 'John', 'age': 30 }");
Console.WriteLine(json["name"]);

NuGet packages are referenced using #r "nuget:PackageName, Version" at the top of the file.

Run the script

dotnet-script script.csx

Pass arguments directly:

dotnet-script script.csx --dry-run
