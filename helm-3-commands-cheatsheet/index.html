Deploy chart from registry

No direct way for the moment. Need to pull, export and then install

helm chart export will create a folder named after the chart (chart.yaml, name), which not necesarily needs to match the repository name where the chart is kept in the registry, although I prefer to make them match. The chart name would be used as EXPORTED_CHART_FOLDER then.
