Build native mobile packages from html/css/js app with PhoneGap Build cloud service.

2020-08-09

General idea

Startup with an html/js/css app. Build an APK from the front-end web app using PhoneGap Build cloud service.
Unfortunatly Phonegap CLI fails for some weird reason, so the app needs to be zipped before posting it to the API (the API doesnâ€™t fail but expects the zip up to where I got).

Phonegap Build account

Create an account in https://build.phonegap.com/.

Build android through dev API

Create PhoneGap Build config file in the app root, e.g., in public folder containing all the html, css, js.

Next zip public folder, containing the app plus config.xml and PUT it for Build to the PhoneGap Build cloud.

PHONEGAP-BUILD-ACCOUNT-USER-ID: the email used as user id for the account created in PhoneGap Build.
APP-ID: the application id. Should be something like 4016017, and is found in https://build.phonegap.com/apps

Provide the password for the account when promped.

Download and install APK

Now, browse to https://build.phonegap.com/apps and use the QR code generated for your app to download APK to the mobile.
